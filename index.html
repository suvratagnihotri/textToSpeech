JEE Advanced Exam Instructions

Read the following instructions carefully before starting the exam:

  1. This exam consists of multiple-choice questions.
  2. You have 3 hours to complete the exam.
  3. There are a total of 20 questions.
  4. Each question carries 4 marks.
  5. There is a negative marking of 1 mark for each wrong answer.
  6. Once you start the exam, you cannot pause or restart it.
